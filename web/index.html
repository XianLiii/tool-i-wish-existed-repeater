Add a new book

    Adding a book is a command-line workflow (no in-browser upload yet). Two modes:

    Text + Audio — best quality, uses your canonical text as ground truth:

    bash scripts/add_book.sh <slug> "<title>" <audio.mp3> <text.md> "<author>"

    Audio-only — fast, no text needed. Whisper transcribes + auto-groups paragraphs by pause:

    bash scripts/add_book.sh <slug> "<title>" <audio.mp3> - "<author>"

    Optional: set CHUNK_SECONDS=1800 to only process the first 30 minutes while testing. Reload this page when done — the new book appears in this menu.

    Transcription speed depends on model. base.en ≈ 1× realtime, large-v3-turbo ≈ 0.2× realtime on Apple Silicon.
